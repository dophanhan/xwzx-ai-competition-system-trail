## 🔑 必需配置项（必须全部填写）

### 1️⃣ FEISHU_APP_ID
- **配置项：** `FEISHU_APP_ID=cli_xxxxxxxxxxxxxx`
- **获取方式：**
  1. 飞书开放平台 → 应用详情
  2. 「凭证与基础信息」标签页
  3. 复制「App ID」（以 `cli_` 开头）
- **示例：** `FEISHU_APP_ID=cli_a90abd1ceff99cc4`

### 2️⃣ FEISHU_APP_SECRET
- **配置项：** `FEISHU_APP_SECRET=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
- **获取方式：**
  1. 飞书开放平台 → 应用详情
  2. 「凭证与基础信息」标签页
  3. 点击「应用密钥」右侧的「查看」或「重置」
  4. 复制密钥（32 位字符串）
- **示例：** `FEISHU_APP_SECRET=oyvAMYAyZnYf76CKMRixbgcnynTO6kPB`

### 3️⃣ FEISHU_BITABLE_APP_TOKEN
- **配置项：** `FEISHU_BITABLE_APP_TOKEN=xxxxxxxxxxxxxxxx`
- **获取方式：**
  1. 打开飞书多维表格（任意一个表格即可）
  2. 复制浏览器地址栏 URL
  3. URL 格式：`https://xxx.feishu.cn/base/`**`AbCdEfGhIjKlMnOp`**`?table=tblxxx`
  4. 复制 `/base/` 后面的部分（16 位字符串）
- **示例：** `FEISHU_BITABLE_APP_TOKEN=PouTdPppvoyMMGx9nICcGcpYn4e`

### 4️⃣ FEISHU_REGISTRATION_TABLE_ID
- **配置项：** `FEISHU_REGISTRATION_TABLE_ID=tbltDHBoAs72cFnh`
- **获取方式：**
  1. 打开【大赛报名表】多维表格
  2. 查看 URL 中的 `?table=` 参数
  3. 或右键表格标签 → 复制表格 ID
- **已确认：** `tbltDHBoAs72cFnh` ✅

### 5️⃣ FEISHU_SUBMISSION_TABLE_ID
- **配置项：** `FEISHU_SUBMISSION_TABLE_ID=tblxYsQVComD9wRW`
- **获取方式：**
  1. 打开【大赛成果提交表】多维表格
  2. 查看 URL 中的 `?table=` 参数
- **已确认：** `tblxYsQVComD9wRW` ✅

## 📬 可选配置项

### 6️⃣ FEISHU_BOT_WEBHOOK（可选，建议配置）
- **配置项：** `FEISHU_BOT_WEBHOOK=https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxxxxxx`
- **获取方式：**
  1. 在飞书中创建一个群聊（或选择现有群）
  2. 点击右上角「设置」→「群机器人」
  3. 点击「添加机器人」→「自定义机器人」
  4. 填写机器人名称（如：AI 大赛通知助手）
  5. 复制 Webhook 地址
- **作用：** 报名和提交成功后，机器人会发送通知到群里
- **不配置的后果：** 系统正常运行，但不会发送飞书通知

### 7️⃣ PORT（可选）
- **配置项：** `PORT=3000`
- **说明：** 服务运行端口，默认 3000
- **修改场景：** 如果 3000 端口被占用，可改为其他端口（如 3001、8080 等）

## 🔐 飞书应用权限配置

创建应用后，需要为应用开通以下权限：

### 必需权限清单

1. **获取租户 access_token**
   - 路径：应用详情 → 权限管理 → 搜索「tenant_access_token」
   - 权限点：`获取租户 access_token`
   - 状态：✅ 必须开通

2. **多维表格数据读写**
   - 路径：应用详情 → 权限管理 → 搜索「bitable」
   - 权限点：`获取或更新多维表格记录`、`获取多维表格应用信息`
   - 状态：✅ 必须开通

3. **发送消息到群组**
   - 路径：应用详情 → 权限管理 → 搜索「message」
   - 权限点：`发送消息到群组`
   - 状态：✅ 必须开通（如果配置了机器人）

### 权限开通步骤
1. 应用详情 → 权限管理
2. 点击「申请权限」
3. 搜索上述权限名称
4. 勾选后点击「提交申请」
5. 等待管理员审批（通常立即生效）

## ❗ 常见错误排查

### 错误 1：获取租户 access_token 失败
- **原因：** FEISHU_APP_ID 或 FEISHU_APP_SECRET 错误
- **解决：** 检查配置是否正确，确保应用已开通权限

### 错误 2：数据写入飞书多维表格失败
- **原因 1：** FEISHU_BITABLE_APP_TOKEN 错误
- **原因 2：** 表格 ID 错误
- **原因 3：** 应用没有多维表格权限
- **解决：** 逐一检查配置，确认权限已开通

### 错误 3：机器人消息发送失败
- **原因：** Webhook 地址错误或已失效
- **解决：** 重新创建机器人，获取新的 Webhook 地址
