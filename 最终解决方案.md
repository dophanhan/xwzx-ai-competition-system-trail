| `drive/v2/media/upload_all` | ❌ 404 | Not Found |

### 结论

**飞书云文档 API 无法通过直接调用 REST API 使用**

可能的原因：
1. 需要使用飞书官方 SDK
2. API 需要特殊的认证方式
3. 企业限制了云文档上传功能
4. API 文档与实际实现不一致
