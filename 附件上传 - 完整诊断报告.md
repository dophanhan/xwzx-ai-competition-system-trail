## 🔍 错误分析

### 错误代码：1061002

**错误信息：** `params error.`

**官方排查链接：**
https://open.feishu.cn/search?from=openapi&log_id=20260306212839CBB2B6F20F94828CB40A&code=1061002

**可能原因：**

1. **参数位置错误**
   - 参数应该在 body 中，而不是 query params
   - 或者参数应该在 headers 中

2. **参数格式错误**
   - `parent_type` 或 `folder_token` 的值不正确
   - 需要其他必需参数

3. **应用权限问题**
   - 虽然有 token，但权限可能不完整
   - 或者权限未生效

4. **API 版本问题**
   - 当前 API 可能已废弃
   - 需要使用新版本 API

5. **企业限制**
   - 企业可能禁用了云文档上传功能
   - 或者需要额外配置

**步骤：**

1. **注册阿里云账号**
   - https://www.aliyun.com

2. **创建 OSS Bucket**
   - 选择地域
   - 设置访问权限（私有）
   - 获取 AccessKey

3. **安装 OSS SDK**
   ```bash
   npm install ali-oss
   ```

4. **修改上传代码**
   ```javascript
   const OSS = require('ali-oss');
   
   const client = new OSS({
       region: 'oss-cn-beijing',
       accessKeyId: 'your access key',
       accessKeySecret: 'your access secret',
       bucket: 'your bucket name'
   });
   
   async function uploadFile(filePath, fileName) {
       const result = await client.put(fileName, filePath);
       return result.url;
   }
   ```

5. **将 URL 写入飞书表格**
   - 使用文本字段存储文件 URL
   - 用户可以点击链接下载
