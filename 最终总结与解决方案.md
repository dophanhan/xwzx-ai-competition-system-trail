## 🔬 完整测试结果

### 测试 1: 上传素材到云文档 API

**API:** `POST /drive/v1/medias/upload_all`

**文档说明：**
- 用途：将文件上传到**云文档中**（如插入到文档作为附件）
- 必需参数：
  - `file_name`: 文件名
  - `parent_type`: 上传点类型（`doc_image`, `docx_image`, `doc_file`, `docx_file`, `sheet_image`）
  - `parent_node`: **云文档的 token**（不是 folder_token）
  - `size`: 文件大小
  - `file`: 文件内容

**问题：**
- 这个 API 是上传到**云文档内部**，不是上传到**云空间**
- 需要一个云文档（如 docx）的 token 作为 parent_node
- 不适用于我们的场景（上传独立文件到云空间）

**结果：** ❌ 不适用

---

### 测试 2: 上传文件到云空间 API

**测试的 API：**
- `POST /drive/v1/files` → 404
- `POST /drive/v1/files/upload` → 404
- `POST /drive/v1/folders/{folder_token}/files/upload` → 404
- `POST /drive/v1/media/upload_all` → 404
- `POST /drive/v1/medias/upload_all` → 400 params error

**结果：** ❌ 全部失败

---

### 测试 3: 飞书官方 SDK

**SDK:** `@larksuiteoapi/node-sdk`

**使用方式：**
```javascript
await client.drive.media.uploadAll({
    data: {
        file: fs.createReadStream(filePath),
        parent_type: 'docx_file',
        parent_node: 'doccn...'  // 需要云文档 token
    }

```

**问题：** SDK 底层也是调用同样的 REST API，同样需要云文档 token

**结果：** ❌ 不适用（需要云文档 token）

---

## 🎯 结论

**飞书云文档 API 无法用于我们的场景：**

1. **上传素材 API** (`medias/upload_all`)
   - 只能上传到云文档内部
   - 需要云文档 token，不能上传到云空间

2. **上传文件 API** (`files/*`)
   - 所有尝试都返回 404
   - API 可能不存在或需要特殊配置

3. **根本原因**
   - 飞书的云文档体系分为：
     - **云空间**：存储独立文件
     - **云文档**：docx、sheet 等，内部可以插入素材
   - 我们的需求是上传到云空间，但 API 只支持上传到云文档

## ✅ 可行解决方案

### 方案 A: 使用阿里云 OSS（推荐）

**优点：**
- ✅ 稳定可靠
- ✅ 文档完善
- ✅ 成本低（约 0.12 元/GB/月）
- ✅ 立即可用

**实施步骤：**

1. **注册阿里云账号**
   - https://www.aliyun.com

2. **创建 OSS Bucket**
   - 地域：选择离用户近的（如华北 2-北京）
   - 权限：私有
   - 获取 AccessKey ID 和 AccessKey Secret

3. **安装 SDK**
   ```bash
   npm install ali-oss
   ```

4. **修改代码**
   ```javascript
   const OSS = require('ali-oss');
   
   const client = new OSS({
       region: 'oss-cn-beijing',
       accessKeyId: process.env.OSS_ACCESS_KEY_ID,
       accessKeySecret: process.env.OSS_ACCESS_KEY_SECRET,
       bucket: process.env.OSS_BUCKET_NAME
   });
   
   async function uploadFile(filePath, fileName) {
       const result = await client.put(`competition/${fileName}`, filePath);
       return result.url;  // 返回文件 URL
   }
   ```

5. **修改表格字段**
   - 将"附件文件"字段改为文本类型
   - 存储文件 URL

**成本估算：**
- 存储费：0.12 元/GB/月
- 流量费：0.50 元/GB（内网免费）
- 请求费：0.01 元/万次

假设 100 个队伍，每个队伍上传 10MB 文件：
- 总存储：1GB
- 月成本：约 0.12 元
